Description du cours :

Ce chapitre se concentre sur le Langage d'Interrogation de Données (LID) en SQL, qui permet de récupérer et manipuler les données stockées dans les bases de données relationnelles. Nous explorerons les commandes SQL essentielles pour effectuer des sélections, des jointures et des sous-requêtes.

Objectifs d'apprentissage :

  • Comprendre les principes du Langage d'Interrogation de Données (LID) en SQL.
  • Apprendre à utiliser la commande SELECT pour interroger les données.
  • Maîtriser les différents types de jointures pour combiner les données provenant de plusieurs tables.
  • Savoir formuler et utiliser des sous-requêtes pour des requêtes plus complexes et dynamiques.
