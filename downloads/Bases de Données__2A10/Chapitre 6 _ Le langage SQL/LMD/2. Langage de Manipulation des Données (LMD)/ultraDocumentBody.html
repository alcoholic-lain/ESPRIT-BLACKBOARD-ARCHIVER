Description du cours :

Ce chapitre se concentre sur le Langage de Manipulation de Données (LMD) en SQL, qui permet d'interagir avec les données au sein des bases de données. Nous explorerons les commandes SQL essentielles pour insérer, mettre à jour et supprimer des données.

Objectifs d'apprentissage :

  • Comprendre les principes du Langage de Manipulation de Données (LMD) en SQL.
  • Apprendre à insérer, mettre à jour et supprimer des données dans une base de données.
