Clé étrangère: attribut dans une relation R qui est clé primaire d’une relation R’

Produit(numPd, designPd, puPd, #numFr)


Contrainte d’intégrité: une clause permettant de contraindre la modification de tables par l’intermédiaire de requêtes d’utilisateurs afin que les données saisies dans la base soient conformes aux données attendues.

Les clés primaires et secondaires sont des contraintes d’intégrité.
